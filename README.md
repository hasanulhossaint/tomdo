> A minimalist, offline-ready, and privacy-focused Kanban board that runs entirely in your browser. No servers, no logins, no trackingâ€”just your tasks.

TOMDO is a modern take on the classic Kanban board, built with pure HTML, CSS, and JavaScript. It's designed for individuals who need a fast, simple, and visually organized way to manage tasks without the overhead of complex software. All data is stored locally in your browser, ensuring your information remains private and accessible even when you're offline.

## ğŸš€ Getting Started

No server or complex build steps required! Get a local copy up and running in minutes.

### Prerequisites

All you need is a modern web browser like Chrome, Firefox, or Safari.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/hasanulhossaint/tomdo.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd tomdo
    ```
3.  **Open `index.html` in your browser:**
    Simply open the `index.html` file directly. That's it!

## âŒ¨ï¸ Usage & Shortcuts

-   **Drag & Drop:** Click and hold a task card to drag it to another column.
-   **Mark as Complete:** Click the checkbox on a task to mark it as done. This will also move it to the "Done" column.
-   **Edit Task:** Double-click any part of a task card to open the inline editor.
-   **Filter Tasks:** Use the search bar at the top to instantly filter the board.
-   **Toggle Theme:** Click the â˜€ï¸/ğŸŒ™ icon to switch between light and dark modes.

## â¬†ï¸â¬‡ï¸ Import / Export

-   **Export:** Click the "Export" button to download a `tomdo-tasks.json` file containing all your current tasks. This is great for backups.
-   **Import:** Click the "Import" button and select a valid `tomdo-tasks.json` file to restore your board from a backup.
